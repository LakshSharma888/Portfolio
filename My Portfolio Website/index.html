Work

This project is a full-featured real estate property selling platform built using the MERN stack (MongoDB, Express.js, React.js, and Node.js) with Prisma as the database ORM. The platform allows users to list, browse, and purchase properties seamlessly. It includes features such as user authentication, property filtering, and detailed property pages with dynamic data fetching.

Prisma’s integration ensures efficient database queries and management, while React provides a smooth, responsive user interface. This application is designed to offer a streamlined and scalable solution for buying and selling real estate properties online.
